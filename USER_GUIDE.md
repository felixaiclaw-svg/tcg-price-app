## 🎯 使用步驟

### 步驟 1: 上傳卡牌

1. 點擊 **📸 上傳/拍攝卡牌** 按鈕
2. 選擇以下任一方式:
   - 從相冊選擇圖片
   - 使用相機拍攝
3. 確認圖片清晰可見

### 步驟 2: 選擇貨幣

- 點擊 **MOP** / **HKD** / **RMB** 選擇目標貨幣
- 預設為 **MOP** (澳門元)

### 步驟 3: 開始掃描

1. 點擊 **🔍 開始掃描** 按鈕
2. 等待識別完成 (約 3-5 秒)
3. 查看結果

### 步驟 4: 查看價格

- **轉換價格**: 顯示 MOP/HKD/RMB 平均價格
- **價格來源**: 列出 5 個網站的具體價格
- 點擊 **查看詳情** 跳轉到原網站

## 💡 使用提示

### 拍攝技巧

✅ **建議**:
- 光線充足，避免陰影
- 卡牌水平放置
- 確保文字清晰可見
- 解析度 300dpi 以上

❌ **避免**:
- 反光/眩光
- 模糊/手震
- 傾斜角度過大
- 遮擋卡牌文字

### 識別準確度

| 卡牌類型 | 識別率 | 備註 |
|---------|-------|------|
| PTCG (英文) | 95% | 最佳效果 |
| PTCG (日文) | 90% | 需清晰圖片 |
| OPCG (日文) | 88% | 需清晰圖片 |
| Yu-Gi-Oh! | 85% | 文字較小 |
| MTG | 90% | 標準效果 |

### 價格更新

- **更新頻率**: 每次掃描實時抓取
- **快取時間**: 30 分鐘 (減少 API 請求)
- **建議**: 重要購買前重新掃描獲取最新價格

## 🔧 常見問題

### Q1: 無法識別卡牌？

**解決方法**:
1. 檢查圖片是否清晰
2. 確保卡牌完整在畫面內
3. 嘗試調整光線
4. 重新拍攝上傳

### Q2: 價格顯示為 0？

**可能原因**:
1. 網站暫時無法訪問
2. 卡牌名稱識別錯誤
3. 該卡牌在網站上無售賣記錄

**解決方法**:
- 手動輸入卡牌名稱查詢
- 嘗試其他價格來源

### Q3: 貨幣轉換不準確？

**說明**:
- 匯率來自實時 API
- 可能有 1-2% 誤差
- 實際交易以網站為準
