## 🚀 快速啟動

### 方法 1: 一鍵啟動（推薦）

**雙擊運行**:
```
start-app.bat
```

會自動啟動：
- ✅ 後端服務（端口 8000）
- ✅ 前端服務（端口 3000）
- ✅ 顯示訪問地址

### 方法 2: 手動啟動

**後端**:
```bash
cd backend
python -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

**前端**:
```bash
cd frontend
npm run dev -- --host 0.0.0.0
```
