## 🌐 外網訪問（可選）

如果需要在外面也能訪問（不在同一 WiFi）：

### 使用 ngrok（簡單）

```bash
# 安裝 ngrok
choco install ngrok

# 啟動內網穿透
ngrok http 3000
```

會生成一個公網地址，任何地方都能訪問。

## ⚠️ 常見問題

### Q1: 手機無法訪問？

**檢查**:
1. 手機和電腦是否同一 WiFi？
2. 防火牆是否允許 3000 端口？
3. 服務是否運行中？

**解決**:
```bash
# 檢查服務
curl http://localhost:3000

# 重啟服務
start-mobile.bat
```

### Q2: 防火牆阻擋？

**Windows 防火牆設置**:
1. 控制面板 → Windows Defender 防火牆
2. 高級設置 → 入站規則
3. 新增規則 → 端口 3000, 8000
4. 允許連接
